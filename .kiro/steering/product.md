# Stock Portfolio Management Software

## Product Overview

A fully offline desktop application for managing stock portfolios for Indian retail investors. The application runs locally on Windows and Mac, with all data stored locally for complete privacy and offline access.

## Target Users

- Indian retail stock investors
- Users who want complete data privacy (no cloud dependency)
- Investors who need accurate capital gains calculations for tax purposes
- Users trading on NSE/BSE through ICICI Breeze

## Key Features

- **Portfolio Management**: Track stocks, transactions, and real-time prices
- **Capital Gains Calculation**: Automated FIFO-based STCG/LTCG calculations
- **Tax Reporting**: Generate financial year-based capital gains reports
- **Real-time Data**: Integration with ICICI Breeze API for live prices
- **Complete Privacy**: All data stored locally, no cloud sync

## Indian Market Specifics

- Financial Year: April 1 to March 31
- Tax Rules: STCG (15%), LTCG (10% above ₹1 lakh)
- Number Formatting: Indian numbering system (lakhs/crores)
- Market Hours: 9:15 AM - 3:30 PM IST
- Currency: Indian Rupee (₹) formatting

## Success Criteria

- Accurate FIFO-based capital gains calculation
- Real-time price updates during market hours
- Generate tax reports for any financial year
- Runs smoothly on Windows 10+ and macOS 10.14+
- Complete offline functionality (except API calls)
- Data remains private and secure on local machine
